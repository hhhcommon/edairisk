信息服务协议

本金融信息服务协议（以下简称“本协议”）由下列各方通过网络点击方式签署，合同签订地为杭州市萧山区。

甲方（借款人） ：借款申请人本人（详见借款申请页）

乙方（出借人）：详见驿贷指定的金融信息服务平台（简称“合作平台”）对应的出借人（以实际出借人为准）

丙方：浙江驿贷网络科技有限公司（简称“驿贷”）

法定代表人：管海闽

住所：浙江省杭州市萧山区湘湖金融小镇二期中区块南岸3号楼322室

丙方是一家在中国杭州市合法成立并有效存续的有限公司，提供金融服务外包、企业管理咨询等服务。甲方（借款人）同意通过驿贷的金融信息服务，与乙方（出借人）达成借款协议并进行融资。

在本协议有效期内向甲方（借款人）提供信息搜集、信息公布、信息交互、金融服务外包等服务，甲方（借款人）向驿贷支付费用的相关事宜达成协议如下：

第一条 名词释义

除本协议另有规定外，下列用语或术语应当具有以下定义：

1.1 出借人：给借款人提供资金的资金方。

1.2 借款总额：借款人通过与出借人签署的《借款合同》拟借入的资金总金额。

第二条 双方主要的权利义务

2.1 借款人的主要权利义务

2.1.1 甲方（借款人）有权根据本协议，享有驿贷提供的服务，同时遵守驿贷公示的规则。

2.1.2 甲方（借款人）通过驿贷提供的服务与乙方（出借人）形成借贷关系时按照本合同约定的签约方式与甲方（出借人）签署合同，并按与甲方（出借人）签署的合同履行相应的义务。

2.1.3 甲方（借款人）保证在接受驿贷服务时不得有以下行为：：

(1)向驿贷及乙方（出借人）提供欺诈性质的借款信息；

(2)同时通过多个网络借贷信息中介机构，或者通过变换项目名称、对项目内容进行非实质性变更等方式，就同一融资项目进行重复融资；

(3)在驿贷平台以外的公开场所发布同一融资项目的信息；

(4)将从驿贷平台的借款用于任何非法用途。

2.1.4 甲方（借款人）确认本合同附件《借款合同》的全部内容，该《借款合同》经乙方（出借人）确认后即对甲方（借款人）具有完全的效力，甲方（借款人）承诺按《借款合同》的内容履行义务。

2.1.5 甲方（借款人）指定的收款银行账户信息详见借款人申请借款文件或与合作平台签署的其他文件的约定，未书面约定的，以借款人实际收到的账户为准。

2.1.6 甲方（借款人）有义务提供自己的真实资料，并保证内容真实、合法、有效、全面和完整，符合中国各项法律法规及规范性文件以及驿贷相关规则的规定并及时更新、维护和管理。甲方（借款人）应承担因信息虚假或无效而产生的任何损失或增加的任何费用等不利后果。

2.1.7 甲方（借款人）承诺已仔细阅读并确认本协议及协议附件《借款合同》等其他驿贷的规则和（或）有关文件的内容。

2.2 丙方（驿贷）的主要权利义务

2.2.1 驿贷有权向甲方（借款人）收取因为借款人与出借人提供金融服务外包、信用咨询等系列相关金融信息服务而由借款人支付给驿贷的报酬。

2.2.2 驿贷有权基于运作以及交易便捷、安全的需要，将部分或者全部服务委托给驿贷的第三方合作机构提供。

2.2.3 驿贷有权要求借款人提供应披露的信息，并对其进行形式审查，驿贷有权根据借款人提供的各项信息（如有保证人、回购人，则包括保证人、回购人提供的信息）决定是否将借款人的信息对接给乙方（出借人）。

2.2.4 驿贷作为信息服务平台，仅以借款人提供的相关的文件、信息为准进行形式审查和核准，并将借款人信息对接给出借人。因借款人提供的信息不全面、不真实、不完整等非因驿贷原因造成的一切不利后果，驿贷不承担任何法律责任。

2.2.5 驿贷向借款人提供信息服务时，驿贷有权根据有关文件、协议和（或）平台页面的相关规则、说明等向借款人收取信息服务费、管理费用和（或）其他费用。借款人与出借人是否成功订立合同，不影响驿贷对上述费用的收取。

2.2.6 驿贷仅为借款人与出借人的信息服务平台，借款人与出借人之间的任何法律纠纷与驿贷无关，除非驿贷与出借人另有书面约定。

2.2.7 驿贷提供的服务涉及与第三方支付机构或金融机构的合作。借款人同意：

(1)受第三方支付机构或金融机构可能仅在工作日进行资金代扣及划转的现状等各种原因所限，驿贷不对服务的资金到账时间做任何承诺，也不承担与此相关的责任，包括但不限于由此产生的利息、货币贬值等损失。

(2)一经使用前述服务，即表示借款人不可撤销地授权驿贷进行相关操作，且该操作是不可逆转的，借款人不能以任何理由拒绝付款或要求取消交易。就前述服务，借款人与第三方支付机构或金融机构之间就费用支付事项产生的争议或纠纷，与驿贷无关。

2.2.8 驿贷有权基于交易安全等方面的考虑不时设定涉及交易的相关事项，包括但不限于交易限额、交易次数等，借款人对此没有异议。

2.2.9 任何情况下，对于借款人在接受驿贷服务过程中涉及的由第三方提供相关服务导致的责任由该第三方承担，驿贷不承担该等责任。因借款人自身的原因导致驿贷无法提供服务或提供服务时发生任何错误而产生的任何损失或责任，由借款人自行负责，驿贷不承担责任。

第三条 借款信息、居间服务费及保证金

3.1 借款信息

3.1.1 借款人明确的借款信息详见借款人申请借款文件或与合作平台、出借人等签署的其他文件的约定。

3.2 信息服务费

3.2.1信息服务费是指驿贷基于本协议向借款人提供平台服务时借款人应向驿贷支付的费用。

3.2.2 本次信息服务费如下：详见借款人申请借款文件或与合作平台、驿贷、出借人等签署的其他文件的约定。

3.2.3 借款人同意在借款成功时根据借款类型的不同向驿贷支付信息服务费等费用，此笔费用借款人和驿贷均同意驿贷在借款成功时从借款本金中直接扣除并转付给驿贷或其指定的第三人。

3.2.4 驿贷通过线上链接服务或线下信息服务等方式帮助借款人向第三方合作平台完成融资信息对接、撮合、咨询等事项的，即视为驿贷为借款人完成了金融信息服务，借款人应当按照相关协议的约定承担信息服务费用及其他约定费用。驿贷有权将本协议项下的权利义务转让给第三方，无需另行征得借款人的同意。未经驿贷书面同意，借款人不得将本协议项下的权利义务转让给第三方。

3.3 保证金

3.3.1 3.3.1借款人应向驿贷支付保证金，保证金不计息，计算公式为：每笔保证金=每笔借款本金× 5 %（具体比例详见借款人申请借款文件或与合作平台、驿贷、出借人等签署的其他文件的约定）。驿贷有权以保证金按利息、罚息、违约金、损害赔偿金、借款本金的顺序冲抵借款人的应付款项；发生直接冲抵情形的，借款人应当在收到驿贷书面通知之日起5日内补足保证金，使保证金余额达到约定之金额，逾期不补足的，借款人应按未补足金额的每日千分之一支付逾期利息。借款人按期足额偿还各项应付款项后，双方清算完毕后7个工作日内退还保证金。 驿贷指定收取保证金的账户信息如下：

第四条 借款合同的签署方式

4.1 甲方（借款人）与乙方（出借人）订立的协议（包括但不限于《借款合同》等的相关规则）可以有一份或者多份并且每一份具有同等法律效力。借款人同意选择以下任一方式签订借款合同：

(1)通过盖章签字的行为确认；

(2)通过网络授权电子合同签章的行为确认；

(3)委托或授权居间信息服务人或居间信息服务人指定的第三方代为确认；

(4)经驿贷认可的其他方式确认。

4.2 借款人以上述任一方式签署协议均视为借款人以本人名义签署了借款合同并代表借款人真实意愿，均具有法律效力。借款人通过前述方式订立的协议对协议各方具有法律约束力，借款人不得否认已订立协议的效力或不按照该等协议约定履行相关义务。

4.3 协议签署后借款人不得擅自修改，驿贷向借款人提供协议的保管、查询、核对等服务。如借款人对协议真伪或内容有任何争议，应以平台记录的协议为准。

4.4 借款人确认，即便在确认借款合同附件时并没有另一方或两方的信息、借款信息等, 也不得以此为由拒绝履行出借人确认的借款合同项下的义务。

4.5信息服务协议对借款合同签订方式的约定与借款合同约定不同的，以借款合同为准。

4.6 驿贷有权根据借款人的授权自行或委托第三方代借款人签署抵押协议、质押协议等相关担保协议和债权转让协议。

第五条 信息披露及保密

5.1 协议任何一方对根据本协议获得的其他方的保密信息（包括但不限于驿贷的技术秘密、资源，借款人的基本资料等保密信息）及本协议所涉保密事项承担同等保密义务，未经协议其他方许可不得擅自将有关信息、资料披露给第三方（根据法律要求履行必要的信息披露义务除外）；如因违反本保密义务给任一方造成经济损失的，应予以赔偿。

5.2 上述保密义务不适用于下列情况：

(1)借款人同意居间信息服务人可使用借款人的个人资料（包括但不限于本平台持有的有关借款人档案中的资料）以提供服务、解决争议、调停纠纷、确保居间信息服务人服务的有效性和平台交易的安全性。

(2)驿贷对于借款人提供的、自行收集到的、经认证的信息将按照有关法律、法规、规范性文件以及本协议及有关规则予以使用或者披露。

5.3 本协议有效期届满，保密条款继续有效。

5.4 借款人注册并在驿贷平台申请借款，为便于金融信息服务的顺利开展，驿贷有权将借款人的融资申请情况及借款人的相关主体信息、风控资信信息、资信审核结果等递交给合作平台进行审核，借款人对此无异议。驿贷及第三方合作平台不得将借款人的相关信息用于融资以外的其他用途。

第六条 违约责任

6.1 本协议生效后，协议双方均应当履行协议约定的义务。任何一方不履行或不完全履行本协议约定的义务的，即构成违约，守约方有权追究违约方的违约责任，并要求赔偿损失，包括但不限于调查费、诉讼费、仲裁费、律师费等。

6.2 因借款人违约或借款人提供的保证人违约，造成驿贷暂停提供服务，给驿贷及出借人造成的所有损失由违约方承担。

6.3 借款人应按本协议之约定向驿贷支付信息服务费用，借款人怠于履行该支付义务的，自借款人违约之日起按照应付信息服务费用的每日万分之八计付违约金；驿贷由此迟延发布或取消发布借款人的相关产品信息而产生的一切后果由借款人承担。

6.4 由于地震、海啸、战争等不可抗力因素造成借款人损失的驿贷不承担违约责任。

6.5 借款人同意,若借款人出现如下任何一种情况,则本协议项下的全部借款自动提前到期,借款人在收到驿贷发出的借款提前到期或解除合同等通知后，应立即清偿全部本金、利息、逾期利息及根据本协议产生的其他全部费用：

6.5.1 借款人因任何原因逾期支付任何一期还款超过3天的

6.5.2 借款人的工作单位、职务或住所变更后，未在30 天内通知驿贷；

6.5.3 借款人发生影响其清偿本协议项下借款的其他不利变化，未在7 天内通知驿贷。

6.6 若借款人逾期90 天支付任何一期还款的，借款人应当按照借款余额的20%支付违约金。

6.7 借款人的每期还款均应按照如下顺序清偿：

6.7.1 根据本协议产生的除本款6.7.2--6.7.6 项之外的其他全部费用；

6.7.2 罚息；

6.7.3 拖欠的利息；

6.7.4 拖欠的本金；

6.7.5 正常的利息；

6.7.6 正常的本金。

第七条争议解决

本协议双方在履行本协议过程中所发生的争议，由双方协商解决。协商不成的，向合同签订地所在地法院提起诉讼。合同签订地为：杭州市萧山区。

8.2 本协议生效后，除本协议已有约定外，本协议任何一方不得擅自变更或提前解除本协议。如确需变更或解除本协议，应经本协议双方协商一致，并达成书面合同。书面合同达成之前，本协议继续执行。
